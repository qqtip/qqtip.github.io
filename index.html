jsish

jsish is a Javascript-like language written in Standard ML and is implemented with type safety and the restrictions of single token lookahead.

It began in a programming languages course abd was a good exercise in bad language design, as the its semantics included many of Javascript's faults.

I genuinely enjoy Javascript and implementing a light clone certainly helped deepen my understanding of the language, but I thought it would be nice to take it a little further. I continued the project by fixing some of Javascript's bad design decisions (like binding variables to the global scope if declared implicitly and the entire concept of the global object) to make a simpler, safer version of the language. I haven't found a use for it yet, but it was fun.

Check it out on GitHub.

Madotsuki Walker

Once upon a time, I hacked together a web-based visual novel by turning DOM elements into menu bars and sprites. This was an attempt to apply this idea to something more dynamic.

Thus came the Madotsuki Walker (sprite/name stolen from Yume Nikki, because I am not an artist), which was an experiment in animating sprite movement in a browser using the DOM and CSS instead of with a canvas. Not my best idea.
